# 立體圖形建構室｜3D Solid Builder

一個給小學生使用的互動式 3D 數學教具，學生可透過新增、刪除、移動「頂點」及連接、刪除「稜」，自由組成不同立體圖形。

## 功能
- 新增／刪除頂點
- 連接／刪除稜
- 拖動頂點改變立體結構
- 360° 旋轉、縮放觀察
- 顯示頂點編號
- 即時計算頂點與稜數量
- 預設正方體、長方體、四面體、四角錐、三角柱、八面體
- 支援電腦、iPad、手機
